Developed a fully functional real-time chat application to enable instant messaging between users.
Implemented user interface using React.js for dynamic, interactive messaging with seamless state management via Redux Toolkit.
Designed a robust backend API using Express.js and MongoDB for storing and retrieving chat messages efficiently.
Integrated WebSocket technology with Socket.IO to enable real-time message updates and notifications.
Optimized application state and API calls using Redux Toolkit's createSlice and createAsyncThunk, improving performance.
